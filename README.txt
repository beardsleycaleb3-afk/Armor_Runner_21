How to publish "Runner-back" on Itch.io
===================================================

1. Go to https://itch.io/game/new (create account if needed)

2. Fill in your game details:
   - Title: Runner-back
   - Kind of project: HTML
   - Upload: Select this ZIP file

3. In "Embed options":
   - Check "Embed in page"
   - Set dimensions: 800 x 600 (or your preference)
   - Enable "Mobile friendly" if your game works on mobile
   - Enable "Automatically start on page load"

4. Click "Save & view page"

Your game is now live on Itch.io!
